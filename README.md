# webpack-react-redux-express #
## 介绍 ##
这是一个搭建好的react、react-router、redux、ant design、express的SPA环境。
## 配置 ##
| 框架           | 版本      
| ------------- |:-------------:
| react         | 15.4.2
| redux         | 3.6.0
| ant design    | 2.7.4
| webpack       | 1.14.0
| express       | 4.15.0 
| react-router  | 3.0.2
## 运行 ##
```
 npm install && npm start
 localhost:4000
```
## 项目入口 ##
+ client
  + src
    + main.js
